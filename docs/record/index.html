Records Api

Let's create New Record For Field.

Route For Create New Recordâ€‹

Provide Full Access Token in Authorization Header
Provide model identifier in URL

https://api-fluidcms.herokuapp.com/record/<modelAlias>

And URL will look like this.

https://api-fluidcms.herokuapp.com/record/demo1

Provide Data in form of key value pairs. and here key refer to field identifier and value is your record

{
    "fields":{
        "name":"husnain",
        "age":"12"
    }
}


In response will look like this.

{
    "ok": true,
    "data": {
        "_id": "617e6858d61e1d0a1719efc2",
        "model_id": "617d796340c7dca02c415916",
        "fields": [
            {
                "field_id": "617da4e3e3683e968b6caf85",
                "value": ""
            },
            {
                "field_id": "617e586fd61e1d0a1719ef48",
                "value": "asdb"
            }
        ]
    }
}

Route Get One Recordâ€‹

Here is single record EndPoint

Providing Record Id in endpoint.
Providing Full access token or Read Only Token.
Provide Access Token in Authorization Header.

https://api-fluidcms.herokuapp.com/record/617e6858d61e1d0a1719efc2

And response will be like this

{
    "ok": true,
    "data": {
        "model_id": "6199807be80d041a69a45014",
        "model_alias": "exp",
        "fields": {
            "name": "husnain",
            "age": "22"
        }
    }
}


Route Get All record Of modelâ€‹

Here is All record EndPoint

You can access single record by Full access token or only Read Token.

Provide Full Access Token in Authorization Header
Provide model identifier in URL

https://api-fluidcms.herokuapp.com/record/<modelAlias>

And URL will look like this.

https://api-fluidcms.herokuapp.com/record/all/demo1

And response will be like this

{
    "ok": true,
    "data": [
        {
            "_id": "617da4e3e3683e968b6caf85",
            "pubg": "good game",
            "Demo": "testing phase",
            "language": "python",

        },
        {
            "_id": "617e586fd61e1d0a1719ef48",
            "pubg": "best",
            "Demo": "testing",
            "language": "java",
        },
    ]
}

Route For Update Recordâ€‹

Here is Record Update EndPoint. Please Provide Full access token in authorization Header to update Record. Provide Record Id in url.

https://api-fluidcms.herokuapp.com/record/617e6858d61e1d0a1719efc2

You can only update Field Value through this endpoint
And Use patch Http method to update User

{
    "fields":{
        "name":"hunny",
        "age":"22"
    }
}

And response will be like this

{
    "ok": true,
    "data": {
        "_id": "617e6858d61e1d0a1719efc2",
        "model_id": "617d796340c7dca02c415916",
        "fields": [
            {
                "field_id": "617da4e3e3683e968b6caf85",
                "value": ""
            },
            {
                "field_id": "617e586fd61e1d0a1719ef48",
                "value": "Hunny"
            }
        ]
    }
}

Route For Deleting Recordâ€‹

Here is Deleting EndPoint through Full access Token. Please Provide Full Access token in authorization Header to Delete Record. Please Provide Record id in URL to Delete Record.

https://api-fluidcms.herokuapp.com/record/617e6858d61e1d0a1719efc2

Response will be like this

{
    "ok": true,
    "data": {
        "message": "Record deleted"
    }
}

Your API Base URL is https://api-fluidcms.herokuapp.com.
